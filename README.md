# Cafeteria do Caio 

## Descrição  
Website institucional moderno e responsivo para uma cafeteria artesanal, criado para apresentar produtos e o ambiente de forma elegante e profissional. A landing page traz um cardápio interativo, seção de contato e um design inspirado em cafeterias boutique, com foco em experiência visual envolvente e alta performance.

## Tecnologias Utilizadas  

- HTML5 semântico  

- CSS3 com animações suaves  

- Design responsivo adaptado para dispositivos móveis, tablets e desktops


## Funcionalidades  

- Layout moderno e clean, com foco na usabilidade  

- Seção de contato simples e objetiva  

- Animações e transições que valorizam a experiência sem comprometer a performance
